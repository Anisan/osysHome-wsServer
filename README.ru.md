# wsServer - Модуль WebSocket сервера

<img src="static/wsServer.png" alt="wsServer Icon" width="50">

WebSocket сервер на базе SocketIO для обеспечения двусторонней связи между сервером и клиентами в реальном времени.

## Описание

Модуль `wsServer` предоставляет полнофункциональный WebSocket сервер для системы osysHome, обеспечивающий коммуникацию в реальном времени между сервером и клиентскими приложениями. Модуль построен на базе Flask-SocketIO и полностью интегрирован с системой объектов osysHome.

## Основные возможности

- ✅ **Подключение/отключение клиентов** с аутентификацией пользователей
- ✅ **Подписки на события**:
    - Изменения свойств объектов (`subscribeProperties`)
    - Изменения объектов (`subscribeObjects`)
    - Выполнение методов (`subscribeActions`)
    - Пользовательские данные (`subscribeData`)
- ✅ **Мониторинг клиентов** с информацией о транспорте и статистике трафика
- ✅ **Управление объектами** через WebSocket:
    - Установка свойств (`setProperty`)
    - Вызов методов (`callMethod`)
- ✅ **Административная панель** для мониторинга подключенных клиентов и событий
- ✅ **LSP поддержка** для редакторов кода

## События и уведомления

Модуль поддерживает следующие типы событий:

### События от сервера к клиенту:

- `say` - отправка текстовых сообщений клиентам
- `notify` - отправка уведомлений
- `playsound` - воспроизведение звуковых файлов
- `changeProperty` - уведомление об изменении свойств объектов
- `changeObject` - уведомление об изменении объектов
- `executedMethod` - уведомление о выполнении методов
- `clients` - информация о подключенных клиентах

### События от клиента к серверу:

- `connect` - подключение клиента
- `disconnect` - отключение клиента
- `subscribeProperties` - подписка на изменения свойств
- `subscribeObjects` - подписка на изменения объектов
- `subscribeActions` - подписка на выполнение действий
- `subscribeData` - подписка на пользовательские данные
- `setProperty` - установка значения свойства
- `callMethod` - вызов метода объекта
- `clients` - запрос списка клиентов
- `lsp` - запросы Language Server Protocol

## Административная панель

Модуль включает административную панель с двумя вкладками:

### Вкладка "Клиенты"

Отображает список всех подключенных клиентов с информацией:

- Имя пользователя (Username)
- IP адрес
- SID (Session ID)
- Время подключения (Connected)
- Тип транспорта (Transport)

### Вкладка "Мониторинг"

Позволяет отслеживать события WebSocket в реальном времени:

- **Пауза/Возобновление** - контроль отображения новых событий
- **Фильтрация событий** - динамические фильтры по типам событий (badge с цветовым выделением)
- **Лог событий** - отображение всех событий с временными метками и данными

## Технические детали

- **Технологии**: Flask-SocketIO, Socket.IO
- **Транспорты**: WebSocket, Long Polling (автоматический fallback)
- **Безопасность**: Аутентификация через Flask-Login
- **Статистика**: Подсчет отправленных/полученных байт для каждого клиента
- **Часовые пояса**: Автоматическая конвертация временных меток с учетом часового пояса пользователя
- **Кэширование**: Кэширование звуковых файлов для воспроизведения

## Использование

### Подключение к WebSocket

```javascript
const socket = io();

socket.on('connect', function() {
    console.log('Connected');
    
    // Подписка на изменения свойств
    socket.emit('subscribeProperties', ['ObjectName.propertyName']);
    
    // Подписка на все свойства
    socket.emit('subscribeProperties', ['*']);
    
    // Подписка на события
    socket.emit('subscribeActions', ['say', 'executedMethod']);
});

// Обработка изменения свойства
socket.on('changeProperty', function(data) {
    console.log('Property changed:', data);
});

// Обработка события say
socket.on('say', function(data) {
    console.log('Message:', data.message);
});
```

### Установка свойства

```javascript
socket.emit('setProperty', 'ObjectName.propertyName', newValue, 'WS');
socket.on('propertyChanged', function(response) {
    if (response.success) {
        console.log('Property updated:', response.property, response.value);
    } else {
        console.error('Error:', response.error);
    }
});
```

### Вызов метода

```javascript
socket.emit('callMethod', 'ObjectName.methodName', 'WS', false);
```
